What are Webhooks by Zapier?

A webhook is an API concept that's growing in popularity. Essentially, a webhook is a way for one application to provide real-time information to another application.

Webhooks can be used by developers to create custom integrations between applications. With Webhooks by Zapier, non-technical users can easily set up these types of integrations without writing any code.

Integration with Zapier
